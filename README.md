 # WeatherCLI# 
WeatherCLI

一个命令行天气查询工具

## 使用方法

1. 获取OpenWeatherMap API密钥
   - 访问 [https://home.openweathermap.org/users/sign_up](https://home.openweathermap.org/users/sign_up) 注册
   - 在控制台获取API密钥

2. 配置环境变量
